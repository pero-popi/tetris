## 遊び方

### ゲームの始め方
- **スタート**  
画面の「START」ボタン
- **リセット**  
画面の「RESET」ボタン
- **ポーズ**  
画面の「PAUSE」ボタン
- **ゴースト**  
画面の「GHOST」ボタン

### プレイヤーの操作方法
- **移動**  
→　右移動  
←　左移動  
↓　通常より速く落下  
↑　ハードドロップ  
- **回転**  
[Z]キー　左回転  
[X]キー　右回転  
- **その他**  
[C]キー　落下中のテトリミノをホールド
